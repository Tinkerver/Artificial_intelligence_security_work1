训练时尝试了使用普通SGD，RMSProp和Adam三种方法

RMSprop使用的时动量的加权平均数，在迭代过程中梯度对当前动量的影响会随着时间慢慢减少。结合当前位置的梯度与过去累积的梯度优化，可以减少波动对整体收敛方向的影响，并加快收敛的速率。

adam在步长选择，动量平均下相对于RMSprop都有优化，利用梯度的一阶矩估计和二阶矩估计动态调整每个参数的学习率，每一次迭代学习率都有一个明确的范围,使得参数变化很平稳。



训练时使用了100次epoch，SGD下loss收敛于0.0049，RMSprop收敛于0.00350，adam收敛于0.00133.在收敛过程中adam与RMSprop的收敛速度都快于SGD，能够在40个epoch时让loss减小到0.01以下。此时adam与RMSprop的差距并不明显。

因此adam在与另两种训练优化器相比时存在优势，可以考虑多使用adam的训练策略。
